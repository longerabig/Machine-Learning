# Machine-Learning
机器学习×机械工程交叉综合项目：基于高级特征工程的航空发动机故障风险预测研究
研究背景与现状：航空发动机故障预测（预测性维护）是航空安全与工业设备运维成本控制的核心问题，当
前机器学习方法虽成为主流，但仍普遍存在特征工程同质化及预测性能与可解释性难以兼顾的问题。
研究内容与创新点：1.数据预处理：基于对数据的全面统计分析，进行缺失值检验、针对传感器数据的平滑
处理以及将连续的剩余使用寿命（RUL）转化为二分类故障标签等常规处理。2.高级特征工程：基于航空发
动机热力学机理与设备退化规律，构造多类能够刻画发动机运行状态与性能衰退的业务驱动特征，而非单
纯通过增加特征数量与模型融合提升预测精度。3.处理类别不平衡问题：在交叉验证框架下对训练折实施
SMOTE等过采样策略。4.模型训练与评估：利用贝叶斯优化对多种主流机器学习模型进行高效调参，并使
用5折交叉验证与阀值搜索技术进一步提升分类精度。5.可解释性分析：从全局与个体两个层面引入SHAP
框架对表现最优的单模型进行可解释性研究，并将SHAP结果用于反向指导特征工程。
最终结果：1.对比实验结果表明，XGBoOst在验证集上达到最优性能，ROC-AUC~0.996，F1-
Score~0.913。2.整合业务驱动特征的XGBoost模型，ROC-AUC相较于仅使用原始特征的基线模型提升
了0.08。3.SHAP分析显示，运行周期数、修正风扇转速、涵道比和高压压气机出口静压是主导故障风险的
最关键驱动因素，验证了发动机随运行时间增加导致热力学性能衰退与核心部件磨损的机械物理逻辑。
